## 👨‍💻 Sobre mí

**Fundador de [CodeCraft Solutions](https://www.codecraftpty.com)**

Construyo software que se usa — sistemas en producción para universidades, agencias de viajes, emisoras de radio y empresas en Panamá. Trabajo en todo el stack: backends web, apps móviles para Android e iOS, bases de datos y despliegues en la nube.

- Desarrollando **Dungeon Life** — dungeon crawler 2D (Flutter + Flame Engine)
- Aplicando principios **OWASP** y desarrollo seguro
- Desplegando en **Vercel**, **Supabase** e infraestructura cloud
- Disponible para **trabajo remoto**, freelance y colaboraciones
